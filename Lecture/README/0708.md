## 지난 시간에 공부한 내용

- Vue 컴포넌트
  - 전역/지역 컴포넌트
  - 컴포넌트 간 통신 (부모/자식 간)
  - 데이터 전달: `props: [...]`
  - 데이터 검증: `props: {...}`
  - 이벤트 방출: `$emit()`, 이벤트 청취: `$on()`
  - 글로벌 이벤트버스 객체를 통한 컴포넌트 통신 (비 부모/자식 간)
  - 이름이 있는 슬롯(Named Slot)
  - 영역을 통해 데이터를 전달하는 슬롯(Scoped Slot)
